📐 A Matemática dos Ângulos Articulares

Para calcular o ângulo em uma articulação B formada pelos pontos A–B–C, o sistema define dois vetores saindo de B:

O ângulo θ é calculado via produto escalar dividido pelo produto das normas (magnitudes):

produto escalar entre os vetores
norma (comprimento) do vetor u

Implementação em Python (NumPy)

def _angle(a, b, c):
    ab = np.array([a[0]-b[0], a[1]-b[1]])  # vetor u = A - B
    cb = np.array([c[0]-b[0], c[1]-b[1]])  # vetor v = C - B

    # Normaliza para vetores unitários
    nab = ab / (np.linalg.norm(ab) + 1e-8)
    ncb = cb / (np.linalg.norm(cb) + 1e-8)

    # Produto escalar → arccos → graus
    cos_ang = np.clip(np.dot(nab, ncb), -1.0, 1.0)
    return float(np.degrees(np.arccos(cos_ang)))
